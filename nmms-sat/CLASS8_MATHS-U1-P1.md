## **1\. Introduction and Definition (அறிமுகம் மற்றும் வரையறை)**

A **Rational Number** is a number that can be expressed in the form of a fraction p/q, where the numerator p and the denominator q are integers and the denominator q is not equal to zero (q \\neq 0).

**விகிதமுறு எண்கள் என்பது p/q வடிவில் உள்ள எண்கள் ஆகும் (p, q முழுக்கள் மற்றும் q \\neq 0).**

The hierarchy of the number system is as follows:

* **Natural Numbers** / **இயல் எண்கள்** (N): 1, 2, 3, 4, ...  
* **Whole Numbers** / **முழு எண்கள்** (W): 0, 1, 2, 3, ...  
* **Integers** / **முழுக்கள்** (Z): ..., \-2, \-1, 0, 1, 2, ...  
* **Rational Numbers** / **விகிதமுறு எண்கள்** (Q): Includes all the above, plus fractions and decimals.

## **2\. Standard Form of Rational Numbers (விகிதமுறு எண்களின் திட்ட வடிவம்)**

According to mathematical standards, a rational number is in its simplest or **Standard Form** if:

1. The denominator is a positive integer.  
2. The numerator and denominator have no common factor other than 1\.

**Example: Reduce 48/-84 to standard form.**

| முறை 1 (Method 1: Step-by-Step) | முறை 2 (Method 2: Using GCD/மீ.பொ.வ) |
| :---- | :---- |
| \\frac{48}{-84} \= \\frac{48 \\div 2}{-84 \\div 2} \= \\frac{24}{-42} | The GCD (மீ.பொ.வ) of 48 and 84 is 12\. |
| \\frac{24 \\div 2}{-42 \\div 2} \= \\frac{12}{-21} | To make the denominator positive and reduce in one step, divide by \-12: |
| \\frac{12 \\div 3}{-21 \\div 3} \= \\frac{4}{-7} | \\frac{48 \\div (-12)}{-84 \\div (-12)} \= \\frac{-4}{7} |
| **Standard Form: \-4/7** | **Standard Form: \-4/7** |

## **3\. Decimal Representation (தசம எண்களாகக் குறித்தல்)**

Rational numbers can be converted into decimals through division. These decimals generally fall into two categories:

* **Terminating Decimals (முடிவுறு தசம எண்கள்):** The division process ends.  
  * *Example:* 1/4 \= 0.25  
* **Non-terminating/Recurring Decimals (முடிவுறா சுழல் தசம எண்கள்):** The division does not end; instead, digits repeat.  
  * *Example:* 1/3 \= 0.333... (represented as 0.\\bar{3})

## **4\. Finding Rational Numbers: Average Method (சராசரி முறை)**

If a and b are any two rational numbers, then the average formula helps find a number situated exactly between them: **c\_1 \= \\frac{1}{2}(a+b)**

**Step-by-Step Practice: Find a number between 1/3 and 5/9.**

1. **Add the numbers:** \\frac{1}{3} \+ \\frac{5}{9} \= \\frac{3}{9} \+ \\frac{5}{9} \= \\frac{8}{9}  
2. **Multiply by 1/2:** \\frac{1}{2} \\times \\frac{8}{9} \= \\frac{4}{9}  
3. **Result:** The rational number 4/9 lies between 1/3 and 5/9.

**Logic (தமிழ் விளக்கம்):** இரண்டு விகிதமுறு எண்களின் சராசரியானது எப்போதும் அந்த இரு எண்களுக்கு இடையில் அமையும்.

## **5\. Basic Operations and Sign Rules (அடிப்படைச் செயல்பாடுகள் மற்றும் குறி விதிகள்)**

### **Sign Rule Cheat Sheet (குறி விதிகள்)**

| Operation | Rule | Result |
| :---- | :---- | :---- |
| Multiplication/Division | (+) \\times (+) or (+) \\div (+) | **(+)** |
| Multiplication/Division | (-) \\times (-) or (-) \\div (-) | **(+)** |
| Multiplication/Division | (+) \\times (-) or (+) \\div (-) | **(-)** |
| Multiplication/Division | (-) \\times (+) or (-) \\div (+) | **(-)** |

### **Division Practice**

**Calculate: 7/-8 \\div \-3/4**

1. Find the **Reciprocal (பெருக்கல் நேர்மாறு)** of the divisor: \-3/4 \\rightarrow 4/-3.  
2. Multiply the first number by this reciprocal: \\frac{7}{-8} \\times \\frac{4}{-3} \= \\frac{28}{24}.  
3. Reduce to Standard Form: \\mathbf{7/6}.

## **6\. Mathematical Properties of Rational Numbers (விகிதமுறு எண்களின் பண்புகள்)**

* **Closure Property (அடைவுப் பண்பு)**  
  * *Rule:* a \+ b or a \\times b results in a rational number.  
  * *Status:* True for Addition, Subtraction, and Multiplication. For **Division**, it is false because dividing by zero is not defined.  
* **Commutative Property (பரிமாற்றுப் பண்பு)**  
  * *Rule:* a \+ b \= b \+ a and a \\times b \= b \\times a.  
  * *Status:* True for Addition and Multiplication. It is **False** for Subtraction (a-b \\neq b-a) and Division (a \\div b \\neq b \\div a) because the order of numbers changes the value.  
* **Associative Property (சேர்ப்புப் பண்பு)**  
  * *Rule:* (a \+ b) \+ c \= a \+ (b \+ c).  
  * *Status:* True for Addition and Multiplication. It is **False** for Subtraction and Division (உண்மையல்ல).  
* **Distributive Property (பங்கீட்டுப் பண்பு)**  
  * *Rule:* a(b \+ c) \= ab \+ ac.  
  * *Status:* Multiplication is distributive over addition and subtraction.

## **7\. Identity and Inverse Elements (சமனி மற்றும் நேர்மாறு உறுப்புகள்)**

* **Additive Identity (கூட்டல் சமனி):** **0**. (a \+ 0 \= a)  
* **Multiplicative Identity (பெருக்கல் சமனி):** **1**. (a \\times 1 \= a)  
* **Additive Inverse (கூட்டல் நேர்மாறு):** The negative of the number. (a \+ (-a) \= 0\)  
* **Multiplicative Inverse / Reciprocal (பெருக்கல் நேர்மாறு):** For a, it is 1/a. (a \\times 1/a \= 1\)

**Zero (0) has no reciprocal** because 1/0 is undefined. This is a very frequent question in the SAT section. **(0-க்கு பெருக்கல் நேர்மாறு கிடையாது).**
